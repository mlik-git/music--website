# 纯静态音乐网站

## 项目结构
```
music/
├── audio
├── templates
│   ├── css
│   ├── js
│   ├── about.html
│   ├── MV.html
│   ├── player.html
│   └── report.html
├── .gitignore
├── index.html
└── README.md
```

## 功能特性
- 响应式音乐播放列表
- 本地音频元数据解析
- 青蓝色系UI风格
- 跨页面播放参数传递

## 技术栈
- HTML5/CSS3
- Vanilla JavaScript
- Python3.10（仅维护）
